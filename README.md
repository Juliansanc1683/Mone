## 🔍 Troubleshooting tips
If a download fails, check the following items:

- Ensure your link is correct. A malformed URL prevents the software from locating the video.
- Check your device storage. You cannot save a file if your phone storage is full.
- Verify your internet connection. A weak connection interrupts the download process.
- Restart the app. Sometimes a quick restart clears temporary connection issues.
